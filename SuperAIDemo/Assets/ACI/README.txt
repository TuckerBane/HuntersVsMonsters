How to incorporate this crafting system create into your existing project:
1. Move the ACI folder into your assets folder.
2. Create an instance of each prefab in the GettingStarted folder
3. Add the component "CraftingComponent" to any objects you want to be part of the crafting system.
4. Open CRAFTING/Recipe Maker and make some crafting recipes.

To test your recipes in game:
5. Press play.
6. Press escape to enter the the in-game menu.
7. Select a recipe from the list under the Set Objective button.
8. Press the set objective button.
